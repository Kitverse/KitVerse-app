# Kitverse 👕

**Tu pasión, camiseta a camiseta.**

Kitverse es una plantilla web para explorar, personalizar y descubrir camisetas deportivas. Pensada como punto de partida rápido para proyectos relacionados con el mundo del fútbol y las camisetas de equipo.

## 🚀 Características

- Estructura simple basada en HTML.
- Fácil de personalizar y extender.
- Ideal como plantilla inicial para catálogos o galerías de camisetas.

## 📂 Estructura del proyecto

```
Kitverse/
└── index.html
```

## 🛠️ Cómo usar este template

1. Haz clic en **"Use this template"** en la parte superior del repositorio.
2. Crea tu propio repositorio a partir de esta base.
3. Clona tu nuevo repositorio:
   ```bash
   git clone https://github.com/tu-usuario/tu-repositorio.git
   ```
4. Abre `index.html` en tu navegador o edítalo según tus necesidades.
